Welcome To Aspect Foundry

The API Exchange Platform.

Collaborate. Exchange. Orchestrate.

Aspect Foundry is a collaborative exchange for developers, businesses, non-profits and public sector organizations to discover and use API's to orchestrate apps and information for clients.

Discover all the tools needed to collaborate with developers, customers and partners while monetizing API's.

Discover other API's inside the exchange to connect with and to create new apps.

Discover the Platform
Aspect Foundry provides all the tools needed to execute the API business model in one place.

Aspect Foundry's API registry continues to grow, and there are no limits to the number of APIs that can be uploaded into the platform.

Create and manage auth keys for APIs and for developers that need to consume APIs.

An integrated CRM provides immediate access to clients and developers with integrated email and messaging.

Keep productivity high while you collaborate with all your teams online using the project management and workflows.

API Registry
Its Simple. Add API's to the registry for search-ability and high visibility.

Aspect Foundry provides one place to find lots of public APIs and Open Data Sources. The registry is visible to all users of the platform.

Search through the registry and use our internal orchestration tools to build new apps.

Done!

Submit your APIs
Aspect Foundry provides several ways to submit APIs.

Submit APIs one at a time and configure end points and reverse proxies, then move on to setting up rates and billing.

Users have the ability to setup reusable template configurations with favorite settings and features.

Aspect Foundry also provides the ability to upload API entry points in bulk, along with URLS pointing to help sites and documentation.

Security and Authentication
Aspect Foundry's integrated OAuth server powers the management, allocation, reset and reconfiguration of auth keys for developers and APIs.

Restrict and manage access control to all API aspects. Using the power of in-memory aspect based access control credential services, true fine grained control over all API entry points is easily possible in real time.

API Rate control and Billing
Manage and set the threshold rates needed to generate revenue at the API level by grouping entry points together based on function, groups and entitlement value.

Manage and adjust rates with fine granularity using ongoing metric thresholds or make rate adjustments in real time.

Aspect Foundry's integrated billing management means developers can make automated payments directly for subscribed API's.

Publish APIs
After configuring API's, make them public by publishing them into the Aspect Foundry API Registry. Opt API's into the Registry to make them searchable to the entire Aspect Foundry exchange community, search engines and more.

Use the integrated CMS system to deploy published APIs to web site's allowing developers to easily sign up and reuse API's and services.

Choose from several site templates and deploy with a few clicks. The published API site comes complete with secure authentication and integrates easily into publishers sites.

API Analytics
Aspect Foundry's integrated analytics helps you to forecast and plan API coverage over time.

Brilliant graphs and charts show where and how subscribers have the most activity in real time, all the time.

Stay ahead and profitable by making adjustments while planning new features and priorities.

Feed analytics data into the project management toolkit and keep teams up to date with roadmaps and project timelines in alignment with SOA Governance models and targets.

SOA Governance
Take advantage of pre-defined templates, plans and reference architectures inside the system library to manage ongoing SOA and API governance efforts.

Enforce API authentication and access control with fine granularity using configurable aspects.

Real time access control is governed using in-memory policy search, request, and decision endpoints with unparalleled speed in the industry.

Ensure that all service entry points have QoS metrics defined and measured in alignment with the reference model.

Encapsulate portfolios of SOA/API projects and resources with increasing and manageable visibility to all stakeholders and clients.

Manage life cycle activities associated with API's with the integrated project management toolkit.
